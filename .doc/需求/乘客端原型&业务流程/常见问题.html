A : 为什么定位不准?

Q : 请检查手机GPS开关是否打开，以获得最准确的定位服务。手机型的定位芯片、操作系统、周围环境、天气原因等都是可能影响您的定位精确程度，但不会影响您的叫车服务，您可以电话联系司机到准确的地点接您。


A : 如何订车?

Q : 您在APP中确定用车时间、上下车地点后点击“预约专车”就可以了


A : 物品遗失在车上如何找回?

Q : 您可以在个人中心一行程记录中找到您的遗失物品行程的司机电话号码，联系司机询问物品信息。


A : 是否可以携带宠物上车?

Q : 根据最新的《网约车管理办法》，乘客不能携带宠物上车，当您携带宠物叫车时，司机有权拒载。
